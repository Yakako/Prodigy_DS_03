# Project Overview

This project implements a Decision Tree Classifier to predict whether a customer will purchase a product or service based on their demographic and behavioral data. The model helps businesses target potential customers more effectively by analyzing patterns in customer behavior.
- The dataset used for this project is the Bank Marketing dataset from the UCI Machine Learning Repository.
